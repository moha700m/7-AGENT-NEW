## 🌐 Deployment
This is a static site. Deploy to Vercel by connecting this GitHub repository.

### Quick Deploy
1. Go to https://vercel.com
2. Click "Add New Project"
3. Import this GitHub repository
4. Click "Deploy"

### Custom Domain
After deployment:
1. Go to Settings → Domains
2. Add `linkarabs.com`
3. Update DNS records in Cloudflare
